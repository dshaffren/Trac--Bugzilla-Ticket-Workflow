Bugzilla ticket workflow for Trac.

see: http://www.bugzilla.org/docs/2.18/html/lifecycle.html

Use trac-admin to add the 'remind' and 'later' resolutions.

Trac [/var/lib/trac/project]> resolution add remind
Trac [/var/lib/trac/project]> resolution add later
Trac [/var/lib/trac/project]> resolution list

Possible Values
---------------
fixed          
invalid        
wontfix        
duplicate      
worksforme     
remind         
later
